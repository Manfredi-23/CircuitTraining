# 7Bit Circuit Training

Adaptive circuit training app for climbers. XP-based progression across 12 muscle groups, form guides, rest timers, session tracking.

## Requirements

- **Node.js** v18+
- **Xcode** (for iOS development — macOS only)

## Quick Start

```bash
npm install
npm run dev
```

Opens the web app at `http://localhost:3000`.

## iOS Development

### First-time setup

```bash
npm install                # Install all dependencies (Next.js, React, Capacitor, etc.)
npm run build              # Compile the app into static files (out/ folder)
npx cap sync ios           # Copy the static files into the iOS Xcode project
npx cap open ios           # Open the Xcode project
```

In Xcode:
1. Select a simulator (e.g. **iPhone 16**) from the device dropdown
2. Press **Cmd+R** to build and run

### After code changes

```bash
npm run build && npx cap sync ios
```

Then **Cmd+R** in Xcode to see the update.

### Run on a physical iPhone

1. Connect your iPhone via USB
2. In Xcode, select your device from the dropdown
3. Go to **Signing & Capabilities** and select your Apple ID as the team
4. Press Cmd+R — Xcode will install the app on your phone
5. On your iPhone: **Settings > General > VPN & Device Management** — trust the developer certificate

No paid Apple Developer account needed for personal testing.

## Vercel Deployment (Web)

The `main` branch auto-deploys to Vercel on push. Preview deploys are created for every PR.

## Project Structure

```
src/
  core/        Pure TypeScript business logic (no React, shared with iOS)
  storage/     Storage abstraction (localStorage now, Supabase later)
  store/       Zustand state management
  hooks/       React hooks (timer, swipe, audio)
  components/  React components + CSS Modules
  app/         Next.js app router (layout, page, globals.css)
ios/           Xcode project (generated by Capacitor)
legacy/        Original vanilla JS version (reference only)
public/        Static assets (images, manifest.json)
```
